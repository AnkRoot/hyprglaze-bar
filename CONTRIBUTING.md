## 🚀 开发环境

### 前置要求

- **Node.js**: 18.0.0+
- **pnpm**: 8.0.0+ (推荐)
- **GlazeWM** 和 **Zebar**: 用于测试

### 快速开始

```bash
# 1. 克隆项目
git clone https://github.com/AnkRoot/hyprglaze-bar.git
cd hyprglaze-bar/zebar/hyprglaze-bar

# 2. 安装依赖
pnpm install

# 3. 开发命令
pnpm dev          # 启动开发服务器
pnpm build        # 构建项目
pnpm preview      # 预览构建结果
```

## 📝 代码规范

### 基本要求

- 使用 TypeScript，避免 `any` 类型
- 遵循 React Hooks 最佳实践
- 使用 TailwindCSS 进行样式编写
- 保持代码简洁易读

### 提交格式

使用约定式提交：`type: description`

**提交类型**:

- `feat`: 新功能
- `fix`: Bug 修复
- `docs`: 文档更新
- `style`: 代码格式调整
- `refactor`: 代码重构
